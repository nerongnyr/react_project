## 💡 프로젝트 소개
인스타그램의 주요 기능을 클론하여 만든 SNS 웹 애플리케이션입니다.
React + Node.js + MySQL 기반으로 개발되었으며, 사용자 인증, 게시물 등록/수정/삭제, 댓글, 팔로우, 알림, 채팅 등 핵심 기능을 구현했습니다.
